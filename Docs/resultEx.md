## 2. IQR 이상치 처리 결과

### 2.1 적용 공식

```text
IQR = Q3 - Q1
하한 = Q1 - 1.5 × IQR
상한 = Q3 + 1.5 × IQR
정상 범위 = 하한 이상, 상한 이하
```

### 2.2 계산 결과

| 항목 | 결과 |
|---|---:|
| Q1 | `[Q1 입력]` |
| Q3 | `[Q3 입력]` |
| IQR | `[IQR 입력]` |
| 정상 범위 하한 | `[하한 입력]` |
| 정상 범위 상한 | `[상한 입력]` |
| 제거 전 행 수 | `[제거 전 행 수]`행 |
| 제거 후 행 수 | `[제거 후 행 수]`행 |
| 제거된 이상치 | `[제거 건수]`건 |

### 2.3 이상치 처리 해석

`amount`가 `[하한 입력]` 이상 `[상한 입력]` 이하인 데이터를 정상 범위로 판단했습니다. 전체 `[제거 전 행 수]`건 중 `[제거 건수]`건을 이상치 후보로 제거했으며, 최종 `[제거 후 행 수]`건을 집계에 사용했습니다.
